# development-work
Contains several utilities that were created during transaction testing and execution analysis. Many scripts were originally temporary tools but later became useful references for development work.

Some examples focus on collecting Key Metrics from execution sessions and preserving information for later inspection. These records help compare different transaction runs.

Several modules also record traded tokens and maintain small summaries that simplify debugging and validation tasks.

The repository additionally includes examples involving multiple blockchain networks. These experiments emphasize interoperability, observation, and repeatable testing procedures.
